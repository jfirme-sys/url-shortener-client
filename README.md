## Tutorial
- Follow the steps to install and run the project:
```bash
# Clone this repository
git clone https://github.com/jfirme-sys/url-shortener-client.git

# Access the project folder
cd url-shortener-client

# Install dependencies
npm install

# Run app!
npm start
```
